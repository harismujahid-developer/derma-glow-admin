# Derma Glow - Administrative Portal

A secure, responsive full-stack administrative web application built to streamline operations, manage inventory, and handle dynamic data for a custom skincare brand.

## 🚀 Tech Stack
* **Backend:** PHP (Object-Oriented / Procedural data routing)
* **Database:** MySQL (Relational schema designed for e-commerce/admin tracking)
* **Frontend:** HTML5, Custom External CSS (Clean, unique layout built without boilerplate frameworks)

## 🛠️ Features
* **Dynamic Administration Dashboard:** Real-time data overview and content management controls.
* **Database Integration:** Secure CRUD operations linked directly to a local XAMPP MySQL deployment.
* **Custom Styling:** Structured architecture emphasizing native layout principles to ensure performance and uniqueness.

## 💻 Setup Instructions
1. Clone this repository or download the source code.
2. Move the project folder into your local server environment directory (e.g., `xampp/htdocs/`).
3. Import the corresponding `.sql` database schema file via **phpMyAdmin**.
4. Configure your database credentials in the central connection file.
5. Launch XAMPP, start Apache and MySQL services, and navigate to `http://localhost/derma-glow-admin/` in your browser.
